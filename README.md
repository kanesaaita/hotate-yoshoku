# ホタテ稚貝養殖 記録・分析ツール（PWA）

サロマ湖のホタテ稚貝養殖向けの、記録・在庫・浮力・育成計算を一気通貫で管理できるWebツールです。

- ブラウザだけで動作。**データは端末内に保存**（外部送信なし）
- **ホーム画面に追加でアプリ化・オフライン起動**（PWA）
- 記録：採苗器投入 → 移動 → 水揚げ → 丸籠育成 → のし管理 → 販売・放流（9工程）
- のし在庫（ちょうちん／丸籠）・🎈浮力管理・🧮育成計算（逆算／配置予定／見込み）

初回は画面の「はじめての方へ」に沿って、自分ののし・目標を登録してください。
公開版にサンプルデータは含まれません（自分のデータは設定→JSON読み込みで移せます）。
